Capítulo 1: Formularios básicos

Práctica 1 - Elementos basicos (8.2 - 8.11)

Para hacer formularios en HTML5, se utilizan los elementos <form> y <input>.

El elemento <form> es el contenedor de los elementos de formulario. Los elementos de formulario son elementos que permiten al usuario introducir datos que serán enviados al servidor para su procesamiento.

El elemento <input> es el elemento que permite al usuario introducir datos. El elemento <input> tiene un atributo type que indica el tipo de datos que se espera que el usuario introduzca. El atributo type puede tener los siguientes valores:

  • text
  • password
  • checkbox
  • radio
  • file
  • submit

El elemento <input> también puede tener un atributo name que indica el nombre que se le asigna al dato introducido por el usuario.
