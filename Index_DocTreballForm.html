Capítulo 1: Formularios básicos

Práctica 1 - Elementos basicos (8.2 - 8.11)

Para hacer formularios en HTML5, se utilizan los elementos <form> y <input>.

En la etiqueta input podemos usar el atributo type con los siguientes valores para decir de que tipo es el dato a introducir:

  • Text: Para introducir texto
  • Password: Para contraseñas (se muestran con puntos)
  • Checkbox: Para marcar una casilla
  • Radio: Para marcar una casilla de un grupo de opciones
  • Submit: Para enviar el formulario
  • Reset: Para borrar el formulario
  • Button: Para crear un botón
  • Hidden: Para crear un campo oculto
  • Image: Para crear un botón con una imagen
  • File: Para subir archivos

Capítulo 2: Formularios avanzados

Práctica 1 - Elementos avanzados (8.13 - 8.16)

Con la etiqueta fieldset podemos agrupar campos en un formulario. Estos campos agrupados podemos darles un título con la etiqueta legend.

Por otro lado, tambien existe el textarea, que actua como input pero tiene su propia etiqueta. El textarea permite introducir texto de varias lineas.

La etiqueta select permite crear un desplegable con opciones. Hay tres tipos de select:

  • El primero es el de las listas más utilizadas que sólo muestran un valor cada vez y sólo permiten seleccionar un valor.
  • El segundo tipo de lista es el que sólo permite seleccionar un valor pero muestra varios a la vez.
  • Por último, el tercer tipo de lista desplegable es aquella que muestra varios valores y permite realizar selecciones múltiples.

Para agrupar opciones en un select, se utiliza la etiqueta optgroup.
