Capítulo 2: Características básicas

Práctica 1 - Primer documento HTML (2.2)

La primera práctica consiste, como su propio nombre indica, en crear el primer documento HTML. Para hacerlo nos ha sido necesario instalar un editor de código (Sublime Text y Visual Studio Code son los más populares) y posteriormente crear un documento HTML con un nombre al azar.

Acto seguido, hemos copiado el siguiente fragmento de código en el documento y hemos abierto el fichero .html en nuestro navegador.

Código:

<html>
<head>
<title> El primer documento HTML </title>
</head>

<body>
El lenguaje HTML es <strong>tan sencillo</strong> que prácticamente se entiende sin estudiar el significado de sus etiquetas principales.</p>
</body>

</html>
Ver práctica

Conocimientos adquiridos con esta práctica:

  • Funcionamiento de las etiquetas
  • Etiquetas html, head, body, title, strong y p
  • Distinción entre el head y el body de un documento HTML

Hay dos tipos de elementos en HTML: los elementos en bloque y los elementos en línea. Estos son distintos por el espacio que ocupan, ya que los elementos en liena ocupan solo el espacio del texto que contienen, mientras que los elementos en bloque ocupan toda el ancho de la pagina.

En la siguiente imagen podemos observar esta diferencia (añadiendo bordes con CSS):

Código:

<html>
<head>
<title> Ejemplo de elementos en línea y elementos de bloque </title>
</head>

Capítulo 3: Texto

Práctica 1 - Los párrafos (3.3)

Podemos escribir párrafos en HTML usando la etiqueta p. Los párrafos son elementos en bloque, por lo que ocupan todo el ancho de la pagina.

Código:

<html>
<head>
<title> Ejemplo de texto estructurado con párrafos </title>
</head>

<body>
<p> Este es el texto que forma el primer párrafo de la página. Los párrafos pueden ocupar varias líneas y el navegador se encarga de ajustar su longitud al tamaño de la ventana. </p>

<p> El segundo párrafo de la página también se define encerrando su texto con la etiqueta p. El navegador también se encarga de separar automáticamente cada párrafo. </p>
</body>

</html>
Ver práctica

Conocimientos adquiridos con esta práctica:

  • Etiqueta p y su uso

Práctica 2 - Las secciones (h) (3.4)

Las secciones son elementos en bloque que sirven para dividir el contenido de una pagina en secciones. Las secciones se definen con las etiquetas h1, h2, h3, h4, h5 y h6. Cada una de estas etiquetas representa una sección distinta, siendo h1 la sección principal y h6 la sección menos importante.

Las etiquetas h solo se usan con el titulo de la sección, y no es necesario ponerla al principio y al final de la sección. Veamos un ejemplo:

Código:

<html>
<head>
<title> Ejemplo de texto estructurado con secciones </title>
</head>

<body>
<h1> Titular de la página </h1>
<p> Párrafo de introducción... </p>
<h2> La primera sub-sección </h2>
<p> Párrafo de contenido... </p>
<h2> Otra subsección </h2>
<p> Más párrafos de contenido... </p>
</body>

</html>
Ver práctica

Conocimientos adquiridos con esta práctica:

  • Secciones
  • Uso de la etiqueta h

Por otro lado también hemos conocido los usos de las etiquetas span y div (mientras hemos estado realizando todo el documento de trabajo).

Capítulo 5: Listas

Práctica 1 - Listas no ordenadas (5.1)

En esta práctica entenderemos el uso de las listas no ordenadas.

Las listas no ordenadas son aquellas que no tienen un orden en particular. Para crear una lista no ordenada se utiliza la etiqueta ul (unordered list). Dentro de esta etiqueta se pueden poner las etiquetas li (list item) para cada elemento de la lista.

Copiamos el siguiente código en nuestro editor de texto y observamos el resultado:

Código:

<html>
<head><title> Ejemplo de etiqueta UL </title></head>
<body>

<h1> Menú </h1>

<ul>
<li> Inicio</li>
<li> Noticias</li>
<li> Artículos</li>
<li> Contacto </li>
</ul>

</body>
</html>
Ver práctica

Conocimientos adquiridos con esta práctica:

  • Funcionamiento de las listas desordenadas

Práctica 2 - Listas ordenadas (5.2)

En esta práctica entenderemos el uso de las listas ordenadas.

Las listas ordenadas son aquellas que tienen un orden en particular. Para crear una lista ordenada se utiliza la etiqueta ol (ordered list). Dentro de esta etiqueta se pueden poner las etiquetas li (list item) para cada elemento de la lista.

Copiamos el siguiente código en nuestro editor de texto y observamos el resultado:

Código:

<html>
<head><title> Ejemplo de etiqueta OL </title></head>
<body>

<h1> Intrucciones </h1>

<ol>
<li>Enchufar correctamente</li>
<li>Comprobar conexiones</li>
<li>Encender el aparato</li>
</ol>

</body>
</html>

Práctica 3 - Listas de definición (5.3)

En esta práctica entenderemos el uso de las etiquetas dt, dl y dd.

Para crear una lista de definición se utiliza la etiqueta dl (definition list). Dentro de esta etiqueta se pueden poner las etiquetas dt (definition term) para cada elemento de la lista. Dentro de esta etiqueta se pueden poner las etiquetas dd (definition description) para cada elemento de la lista.

Copiamos el siguiente código en nuestro editor de texto y observamos el resultado:

Código:

<dl>
<dt> SGML </dt>
<dd> Metalenguaje para la definición de otros lenguajes de marcado_basico_de_texto </dd>

<dt> XML </dt>
<dd> Lenguaje basado en SGML y que se emplea para describir datos. </dd>

<dt> RSS</dt>
<dt> GML</dt>
<dt> XHTML</dt>
<dt> SVG</dt>
<dt> XUL </dt>
<dd> Lenguajes derivados de XML para determinadas aplicaciones </dd>
</dl>

</body>
</html>

Capítulo 7: Tablas

Práctica 1 - Ejemplo de tabla sencilla (7.2)

En esta práctica entenderemos cómo se crean tablas en HTML.

Para crear una tabla en HTML, debemos utilizar la etiqueta table. Dentro de esta etiqueta, debemos incluir la etiqueta tr que indica una fila de la tabla. Por otro lado, dentro de table, debemos usar la etiqueta td para referirnos a cada celda de la tabla.

Replicamos el siguiente código y observamos que se crea una tabla con 3 filas y 3 columnas, a parte de la fila de encima.

Código:

<head><title> Ejemplo de tabla sencilla </title></head>
<body>

<h1> Listado de cursos </h1>

<table>
<tr>
<td><strong> Curso </strong></td>
<td><strong> Horas </strong></td>
<td><strong> Horario </strong></td>
</tr>

<tr>

Práctica 4 - Fusión de columnas (7.5)

Para fusionar dos columnas de la tabla usamos el atributo colspan en la etiqueta td. El número que pongamos dentro del atributo indica cuantas columnas usa esa celda de la tabla.

Replicamos el siguiente código y observamos que se crea una tabla como la de la imagen.

Código:

<table>
<tr>
<td colspan="2"> A </td>
</tr>

<tr>
<td> B </td>
<td> C </td>
</tr>
</table>

Práctica 5 - Fusión de filas (7.6)

Para fusionar dos filas de la tabla usamos el atributo colrow en la etiqueta td. El número que pongamos dentro del atributo indica cuantas filas usa esa celda de la tabla. Es necesario poner la etiqueta td en la primera fila de todas de la altitud de esa celda.

Replicamos el siguiente código y observamos que se crea una tabla como la de la imagen.

Código:

<table>
<tr>
<td> A </td>
<td rowspan="2"> B </td>
</tr>

<tr>
<td> C </td>
</tr>
</table>
